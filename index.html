Basic commands

El comando "echo" en Bash se utiliza para mostrar mensajes o variables en la salida estándar. Puedes usarlo para imprimir texto o el valor de una variable en la pantalla. Por ejemplo, puedes escribir "echo Hola" para imprimir la palabra "Hola" en la consola. 

                        
                          echo hello

El comando "echo" en Bash se utiliza para mostrar mensajes o variables en la salida estándar. Puedes usarlo para imprimir texto o el valor de una variable en la pantalla. Por ejemplo, puedes escribir "echo Hola" para imprimir la palabra "Hola" en la consola. 

                        
                          echo hello

El comando "vim" en Bash se utiliza para abrir y editar archivos de texto en el editor de texto Vim. Cuando ejecutas "vim nombre_archivo.txt", se abrirá el archivo llamado "nombre_archivo.txt" en Vim. Vim es un editor de texto poderoso y altamente configurable que te permite realizar diversas operaciones de edición, como agregar, eliminar o modificar el contenido del archivo.

                        
                            vim nombre_archivo.txt

El comando "cat" en Bash se utiliza para mostrar el contenido de un archivo de texto en la salida estándar. Al ejecutar "cat nombre_archivo.txt", se imprimirá el contenido del archivo "nombre_archivo.txt" en la consola. También se puede utilizar para concatenar o combinar múltiples archivos y mostrar su contenido.

                        
                            cat nombre_archivo.txt

El comando "ls" en Bash se utiliza para listar los archivos y directorios en el directorio actual. Al ejecutar "ls", se mostrará una lista de nombres de archivos y/o directorios en la salida estándar. Puedes usar diferentes opciones con "ls" para obtener información adicional, como permisos, tamaño, fecha de modificación, etc. Por ejemplo, "ls -l" muestra una lista detallada con información adicional.

                        
                            ls

El comando "pwd" en Bash se utiliza para imprimir la ruta del directorio actual (present working directory). Al ejecutar "pwd", se mostrará la ruta completa del directorio en el que te encuentras actualmente. Es útil para saber en qué ubicación estás trabajando dentro de la estructura de directorios.

                        
                            pwd

El comando "bash nombre_archivo.sh" en Bash se utiliza para ejecutar un archivo de script de Bash con extensión ".sh". Al ejecutar esta línea de comandos, se iniciará una nueva instancia del intérprete de comandos Bash y se ejecutará el script contenido en el archivo "nombre_archivo.sh". Este comando es útil cuando se desea ejecutar un archivo de script de Bash de manera explícita.

                        
                            bash nombre_archivo.sh

El comando "echo $SHELL" en Bash se utiliza para imprimir en la salida estándar la ruta del intérprete de comandos utilizado actualmente por el usuario. Al ejecutar "echo $SHELL", se mostrará la ruta completa del intérprete de comandos (por ejemplo, /bin/bash) que se está utilizando para ejecutar los comandos en ese momento. Es útil para saber qué intérprete de comandos se está utilizando en tu entorno.

                        
                            echo $SHELL

El comando "ls -l" en Bash se utiliza para listar los archivos y directorios en el directorio actual en formato detallado. Al ejecutar "ls -l", se mostrará una lista de los archivos y directorios en columnas, con información adicional, como los permisos, el propietario, el tamaño, la fecha de modificación y el nombre del archivo o directorio. Es útil para obtener una visión más detallada de los archivos y directorios en un directorio específico.

                        
                            ls -l

El comando "chmod" en Bash se utiliza para cambiar los permisos de archivos y directorios. Permite modificar los permisos de lectura (r), escritura (w) y ejecución (x) para el propietario, grupo y otros usuarios. La sintaxis básica del comando es: "chmod [permisos] [archivo/directorio]". Para especificar los permisos, puedes utilizar números octales (por ejemplo, 755) o la notación simbólica (por ejemplo, u+rwx, g-w, o +x). Los permisos pueden ser representados por las letras u (propietario), g (grupo), o (otros), o por todos ellos combinados (ugo). Los permisos de lectura, escritura y ejecución se representan con las letras r, w y x, respectivamente. Por ejemplo, "chmod 755 archivo.sh" establecerá permisos de lectura, escritura y ejecución para el propietario y permisos de lectura y ejecución para el grupo y otros usuarios en el archivo "archivo.sh".

                        
                            chmod 755 nombre_archivo.sh

o

                        
                            chmod u+rwx,g+r,o+x nombre_archivo.sh

Esto establecerá los permisos adecuados en el archivo "nombre_archivo.sh". El propietario tendrá permisos de lectura, escritura y ejecución (u+rwx), el grupo tendrá permisos de lectura (g+r) y otros usuarios tendrán permisos de ejecución (o+x). Recuerda que puedes combinar diferentes letras para especificar los permisos para el propietario (u), grupo (g) y otros usuarios (o). Las letras r, w y x representan los permisos de lectura, escritura y ejecución, respectivamente.

Basic commands

El comando "echo" en Bash se utiliza para mostrar mensajes o variables en la salida estándar. Puedes usarlo para imprimir texto o el valor de una variable en la pantalla. Por ejemplo, puedes escribir "echo Hola" para imprimir la palabra "Hola" en la consola. 

                        
                          echo hello

Basic commands

El comando "vim" en Bash se utiliza para abrir y editar archivos de texto en el editor de texto Vim. Cuando ejecutas "vim nombre_archivo.txt", se abrirá el archivo llamado "nombre_archivo.txt" en Vim. Vim es un editor de texto poderoso y altamente configurable que te permite realizar diversas operaciones de edición, como agregar, eliminar o modificar el contenido del archivo.

                        
                            vim nombre_archivo.txt

Basic commands

El comando "cat" en Bash se utiliza para mostrar el contenido de un archivo de texto en la salida estándar. Al ejecutar "cat nombre_archivo.txt", se imprimirá el contenido del archivo "nombre_archivo.txt" en la consola. También se puede utilizar para concatenar o combinar múltiples archivos y mostrar su contenido.

                        
                            cat nombre_archivo.txt
